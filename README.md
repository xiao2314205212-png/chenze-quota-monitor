## 使用说明

**不需要你开电脑。** 监控运行在 GitHub Actions 上，全自动 24/7。

1. 关注飞书群「陈泽网」中的通知
2. 收到 **"🎉 陈泽网名额开放！快抢！"** 时，立刻打开手机浏览器访问 `www.chenze1103.com` 注册
3. 名额检测每 60 秒一次，推送延迟不超过 1 分钟


## 通知类型

| 通知 | 含义 |
|------|------|
| ✅ 监控已启动 | 系统正常运行 |
| 💓 心跳检测 | 每 3 小时一次，确认存活 |
| 🎉 名额开放！快抢！ | 可以注册了，立刻去 |


## 自行部署

1. Fork 本仓库
2. 在 Settings → Secrets 中添加：
   - `FEISHU_APP_ID` — 飞书应用 ID
   - `FEISHU_APP_SECRET` — 飞书应用密钥
   - `FEISHU_CHAT_ID` — 飞书群 chat_id
3. 手动触发 Actions → `workflow_dispatch`


## ## 关于 Supabase Key

代码中的  为陈泽网前端公开的 publishable key，非敏感密钥。
任何人打开陈泽网按 F12 均可获取，属 Supabase 设计上的公开字段。
